# 🐘 backend-zoo

A collection of backend implementations in different languages and frameworks.

## Animals in the zoo

| Animal (Framework) | Language | Habitat (Endpoint) |
|-------------------|----------|--------------------|
| 🐍 FastAPI | Python | `/api/users` |
| 🐱‍👤 NestJS | TypeScript | `/api/posts` |
| 🦫 Gin (coming) | Go | TBD |

## Why a zoo?

Because every framework has its own personality — some are fast and sleek (FastAPI), some are structured and architectural (NestJS), some are minimal and powerful (Gin). Let them all live together.
